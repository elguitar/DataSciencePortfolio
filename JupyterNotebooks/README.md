# Jupyter Notebooks
My jupyter notebooks and other related work

## Installation

This repo utilizes python virtualenvs.

To install a virtualenv, run:

```bash
cd Notebooks
virtualenv venv
```

Then, activate it:

```bash
source ./venv/bin/activate
```

And then install the dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run jupyter:

```bash
jupyter-notebook
```
